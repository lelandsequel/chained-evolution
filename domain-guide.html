Connect chainedevolution.com

Follow the steps for where you bought the domain. Takes 5 minutes.

Where did you buy the domain?

GoDaddy → Namecheap → Google Domains or Squarespace → WordPress.com → I'm not sure →

GoDaddy

1
Go to godaddy.com and log in
2
Click your name in the top right → tap "My Products"
3
Find chainedevolution.com in the list → tap "DNS" next to it
4
In the table, find the row where Type = A and Name = @ → tap the pencil icon to edit it
5
Change the Value to: 76.76.21.21 → tap Save
6
Find the row where Type = CNAME and Name = www → tap the pencil to edit it
7
Change the Value to: cname.vercel-dns.com → tap Save
✅ You're done! Text Leland and let him know — he'll finish it on his end within a few minutes.

Namecheap

1
Go to namecheap.com and log in
2
Click "Domain List" in the left sidebar
3
Find chainedevolution.com → click "Manage"
4
Click the "Advanced DNS" tab at the top
5
Find the row with Type: A Record and Host: @ → click the edit icon → change Value to 76.76.21.21 → save
6
Find the row with Type: CNAME and Host: www → click edit → change Value to cname.vercel-dns.com → save
✅ You're done! Text Leland and let him know — he'll finish it on his end within a few minutes.

Google Domains / Squarespace

1
Go to domains.google.com (or squarespace.com → Domains) and log in
2
Click on chainedevolution.com
3
Click "DNS" in the left menu
4
Scroll down to "Custom Records"
5
Find the A record with host @ → click edit → change value to 76.76.21.21 → save
6
Find the CNAME record with host www → click edit → change value to cname.vercel-dns.com → save
✅ You're done! Text Leland and let him know — he'll finish it on his end within a few minutes.

WordPress.com

1
Go to wordpress.com and log in
2
Click your profile icon in the top right → click "Upgrades" then "Domains"
3
Find chainedevolution.com in the list → click on it
4
Click "DNS Records" in the left menu
5
Look for any existing A record with name @ — if there is one, click the trash icon to delete it first
6
Click "Add DNS Record" → set Type to A, Name to @, Value to 76.76.21.21 → click Add
7
Look for any existing CNAME record with name www — delete it if there is one
8
Click "Add DNS Record" again → set Type to CNAME, Name to www, Value to cname.vercel-dns.com → click Add
✅ You're done! Text Leland and let him know — he'll finish it on his end within a few minutes.

Not Sure?

1
Go to whois.domaintools.com on your phone
2
Type in chainedevolution.com and search
3
Look for a line that says "Registrar" — that's where it was bought
4
Log into that site and look for DNS settings — you're looking for an A record and a CNAME record
5
Still confused? Screenshot whatever you're looking at and send it to Leland. He'll tell you exactly what to tap.
